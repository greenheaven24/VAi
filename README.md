A task Kanban board (`/tasks`) and a polygon image annotation tool
(`/annotate`) behind a single login, built with Next.js + TypeScript on the
frontend and Django + DRF on the backend.

- Backend setup, API contract, and difficulties/solutions: [backend/README.md](backend/README.md)
- Frontend setup, structure, and difficulties/solutions: [frontend/README.md](frontend/README.md)

Quick start: run the backend first (`backend/README.md`), then the frontend
(`frontend/README.md`), and log in at `http://localhost:3000/login` with
**demo@demo.com / DemoPass123!**.
